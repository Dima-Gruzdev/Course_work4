# Система управления email-рассылками

Проект представляет собой **веб-приложение для создания и управления email-рассылками**, с поддержкой пользователей, шаблонов писем, получателей и статистики отправок.

Написан на **Django** с использованием **Poetry** для управления зависимостями.

---

## Функционал

###  Управление клиентами
- Добавление, просмотр, редактирование и удаление получателей
- Поля: Email (уникальный), Ф.И.О., комментарий

### Управление сообщениями
- Создание и редактирование шаблонов писем
- Поля: Тема письма, тело письма

### Управление рассылками
- Настройка даты и времени первой и последней отправки
- Статусы: "Создана", "Запущена", "Завершена"
- Привязка к сообщению и списку получателей

### Отправка вручную
- Возможность запустить рассылку в любое время через интерфейс
- Поддержка командной строки (`python manage.py send_mailing`)

###  Попытки рассылок
- Автоматическая фиксация каждой попытки отправки
- Поля: дата/время, статус (успешно / не успешно), ответ сервера

### Регистрация и аутентификация
- Регистрация с подтверждением email
- Вход и выход из системы

### Статистика и отчёты
- Отображение количества успешных и неудачных попыток
- Общее количество отправленных сообщений

### Ограничение доступа
- Пользователь видит и управляет только своими данными
- Менеджер может просматривать всех пользователей и рассылки

### Кеширование
- Серверное кеширование для повышения производительности
- Клиентское кеширование страниц

### Главная страница
- Отображает:
  - Общее количество рассылок
  - Количество активных рассылок
  - Количество уникальных получателей
